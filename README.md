# 🗳️ Online Voting System (JDBC + PostgreSQL)

A simple Java-based CLI application for voting, using **JDBC** and **PostgreSQL**.  
Users can:
- ✅ Add new candidates
- 🗳️ Vote for existing candidates
- 📋 View all candidates and their vote counts
